Found Text

FoundText is a web application designed for Javascript programmers new to Regular Expressions. Using a simple Drag'N'Drop feature, users can begin to understand how regex symbols correspond to certain characters in a string of text.

  • Tech Stack: React, Redux, Regex
  • Created with 2 other web devlopers uring a 24 hour coding hackathon
  • Programmed text-field to highlight based on selected regular expressions
  • Designed Drag/n/Drop feature to store each regex phrase in order

Sleep Mood

Sleep Mood is a web application that tracks not only your sleep, but also how you feel when you go to sleep each day. Every entry is saved to your specific user account, and after a month Sleep Mood will recommend an amount of sleep to leave you the happiest you can be!
Devin desgined the back-end API using Java/Spring for this application to store all the user and sleep data.

MyDish: Your Recipe Editor

My Dish is a mobile app to help users keep track of their recipes in a brand new way. When you edit a recipe that you've created, you can view previous edits that you've made to a recipe. Maybe you thought adding extra sugar would make a cake taste better, but now you've changed your mind. Go back to the previous version you wrote, and cook from that recipe!
Devin worked with a team of 7 full-stack developers to help build and deploy this React-Native app. He wrote and ran unit tests for the front end and back end codebases, as well as refactoring any code to properly pass each test. Check it out on Test Flight or The Google Play Store.
