Intro

Devin has worked in the corporate world specializing in Adult Learning and professional training. Since then he has expanded his horizons into the computer science field. By crafting code that is easy to decipher, Devin designs web pages that are good-looking, easy to use, and easy to understand under the hood. Click here to learn more.

As a graduate from the Full-Stack Web Development program at Lambda School, Devin has skills in HTML, CSS, Javascript, React, Java-Spring Boot, and Python. View his work here.

Work


Noise Controller

Noise Controller is a web application designed for teachers, so they can keep their classes quiet. This marketing page was designed by Devin to entice users to give noise controller a test.

View the GitHub page with this site's code.



Found Text

FoundText is a webpage designed for Javascript programmers new to Regular Expressions. Using a simple Drag'N'Drop feature, users can begin to understand how regex symbols correspond to certain characters in a string of text.
Devin designed this page with two other developers as a part of Lambda School's two-day hackathon- This project won Best Overall.

View the GitHub page with this site's code.



Sleep Mood

Sleep Mood is a web application that tracks not only your sleep, but also how you feel when you go to sleep each day. Every entry is saved to your specific user account, and after a month Sleep Mood will recommend an amount of sleep to leave you the happiest you can be!
Devin desgined the back-end API for this application to store all the user and sleep data.

View the GitHub page with this site's code.
